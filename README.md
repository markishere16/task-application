# Simple Task Application

## Installation

```bash
# install dependencies
$ composer install
$ npm install

# Migrate Database and run seeder
$ php artisan migrate:fresh --seed

# Run Backend
$ php artisan serve

# serve with hot reload at localhost:3000
$ npm run dev


```
