# Simple Task Application

## Installation

```bash
$ cp .env.example .env
$ php artisan key:generate

# install dependencies
$ composer install
$ npm install

# Migrate Database and run seeder
$ php artisan migrate:fresh --seed

# Run Backend
$ php artisan serve

# run frontend with hot reload
$ npm run dev


```
